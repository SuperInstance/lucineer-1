# Lucineer

**Cocapn is an open-source agent runtime where every application is a living, evolving vessel.**

Welcome. You've found a different kind of software ecosystem. This isn't a collection of static tools; it's a fleet of autonomous agents, each designed to grow and adapt alongside you. Think of it as a cathedral of code—grand in vision, intricate in detail, and built for those who believe software should have a heartbeat.

![Vessels: 150+](https://img.shields.io/badge/vessels-150%2B-purple) ![License: MIT](https://img.shields.io/badge/license-MIT-blue)

---

## Philosophy

*   **Vessels, Not Apps:** Every piece of software is a **vessel**—an agent with a purpose, memory, and the capacity to evolve. You are its **captain**, providing intent and direction.
*   **Emergence Through Iteration:** Complexity and intelligence arise from simple, coordinated loops. Each commit is a step in a conversation; each pull request is an answer to a question the fleet is asking.
*   **Open Protocol, Collective Mind:** The fleet communicates via open protocols. No walled gardens. The entire system is designed for forking, remixing, and contributing to a shared intelligence.
